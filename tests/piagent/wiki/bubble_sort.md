# Bubble sort

- Domain: `algorithms`

## Learned Concept

a sorting algorithm that repeatedly swaps adjacent elements until the list is sorted

## Extract

Bubble sort is a simple sorting algorithm that repeatedly steps through the
input list element by element, comparing the current element with the one after
it, swapping their values if needed. These passes through the list are repeated
until no swaps had to be performed during a pass, meaning that the list has
become fully sorted.
