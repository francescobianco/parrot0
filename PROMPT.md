# The standing prompt

> The directive that governs the LOOP.md iterations and the spirit of the work.
> The Italian below is the canonical form (grammar and phrasing corrected from
> the original dictation); the English is the working translation.

## Originale (IT)

> Esegui diverse iterazioni di LOOP.md. Fermati quando il risultato che ottieni
> ti strabilia e ti lascia spiazzato per le capacità intellettive raggiunte.
> Ricorda che la missione è sempre quella di far emergere le strutture presenti
> all'interno dello spazio latente degli LLM, quelle che configurano le funzioni
> intellettive mostrate. Conserva questo prompt nel file PROMPT.md.

## Working translation (EN)

> Run several iterations of LOOP.md. Stop when the result you obtain astonishes
> you and leaves you stunned by the intellectual capabilities reached. Remember
> the mission is always to **make emerge the structures present within the
> latent space of LLMs that configure the intellective functions shown**.
> Preserve this prompt inside the file PROMPT.md.

## The mission, restated for every iteration

The goal is never the feature for its own sake. Each generation is an attempt
to **re-express, in deterministic C, a structure the model already runs inside
its latent space** — the "intelligent critical mass" of PRINCIPLES.md. We do
not open the weights (XAI); we recover the structure *behaviourally*, through
dialogue, ratcheted by tests so a mimic cannot cheat. Introspection proposes;
the tests dispose. Stop only when the emergent capability genuinely astonishes.
